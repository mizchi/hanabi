# Hanabi

![](http://i.gyazo.com/cb21a8f556bc6e3b5b43f83d235aeec9.gif)

```
bower install hanabi
```

## How to use

```javascript
window.addEventListener('load', function(){
  var hanabi = new Hanabi({x: 100, y: 100});
  hanabi.fire()
});
```

## Options

- quantity
- size
- reduceRate
- gravity
- speed
- color
- width
- height
- x
- y
- parent
